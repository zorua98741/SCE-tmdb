URL Generation

In short, the URL is made up of four components:

  • The domain: https://api.serialstation.com
  • The path: /titles/
  • The "title ID type", aka region: ?title_id_type=ABCD
  • The "title ID number": &title_id_number=12345

Combined, this gives the overall format of https://api.serialstation.com/titles/?title_id_type=[titleIdType]&title_id_number=[titleIdNumber]
